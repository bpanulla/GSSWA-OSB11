The Web:

Links, across the tubes.


Current links:

  • href="..."
  • src="..."







Your data can work the same way.

Id's and id references don't all have to be together in your local custom database, they can span the web.

The loose coupling provided by this allows many sites to use and colaborate together on data sets.

{
  "person_1" : {
    "type" : "Person",
    "name" : "leif",
    "age" : 32,
    "knows" : {
      "type" : "Person"
      "name" : "brian"
      "knows" : ...
     }
  }

  "person_2" : {
    "name" : "brian"
  }
   
}
<p>
  <span property="name">Leif</span>
  <a rel="knows" href="http://brian.com/">
    <span property="name">Brian</span>
  </a> 
</p>
